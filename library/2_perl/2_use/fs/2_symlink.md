symlink - создание символьных ссылок
    symlink(filename, linkname)

Описание
========

Функция создает символьную ссылку на файл.

Аргументы
=========

* filename - имя целевого файла;
* linkname - имя ссылки.

Пример использования
====================

Создание ссылки на файл:

    symlink('file.txt', 'link');
