link - создание жестких ссылок
    link(filename, linkname)

Описание
========

Функция создает жесткую ссылку на файл.

Аргументы
=========

* filename - имя целевого файла;
* linkname - имя ссылки.

Пример использования
====================

Создание ссылки на файл:

    link('file.txt', 'link');
