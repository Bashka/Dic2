chr - получение символа по ее коду
    chr(num)

Описание
========

Функция возвращает символ, представленный данным числом.

Аргументы
=========

* num - код целевого символа.

Возвращаемое значение
=====================

Символ, соответствующий данному коду.

Пример использования
====================

Получение символа A:

    print chr(65);
