sin - вычисление синуса угла
    float sin(float arg)

Описание
========

Функция вычисляет синус угла.

Аргументы
=========

* arg - угол в радианах.

Возвращаемое значение
=====================

Синус угла.

Пример использования
====================

Вычисление синуса угла 60 градусов:

    echo sin(deg2rad(60)); // 0.866...
