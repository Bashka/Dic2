end - смещение указателя массива в конец
    mixed end(array &arr)

Описание
========

Функция смещает внутренний указатель массива на его последний элемент.

Аргументы
=========

* arr - целевой массив.

Возвращаемое значение
=====================

Значение последнего элемента или False, если массив пуст.
