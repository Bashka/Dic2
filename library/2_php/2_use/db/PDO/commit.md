commit - запись изменений транзакции.
    bool PDO::commit()

Описание
========

Метод записывает все сделанные в последней транзакции изменения.

Возвращаемое значение
=====================

Возвращает True в случае успешного завершения и False в случае ошибки.

Пример использования
====================

Выполнение транзакции и запись изменений:

    $db = new PDO('mysql:dbname=test;host=localhost', 'root', '123');
    $db->beginTransaction(); // Начало транзакции.
    $db->query('DELETE FROM myTable WHERE id < 10');
    $db->commit(); // Применение изменений.
