date - форматирование временной метки
    string date(string format, [int timestamp])

Описание
========

Функция представляет временную метку в текстовом формате.

Аргументы
=========

* format - формат метки в виде строки. Сюда могут входить следующие флаги:
    * s - секунды с ведущими нулями (от 00 до 59);
    * i - минуты с ведущими нулями (от 00 до 59);
    * g - часы в 12-часовом формате без ведущих нулей (от 1 до 12);
    * G - часы в 24-часовом формате без ведущих нулей (от 0 до 23);
    * h - часы в 12-часовом формате с ведущими нулями (от 01 до 12);
    * H - часы в 24-часовом формате с ведущими нулями (от 00 до 23);
    * w - порядковый номер дня недели (от 0 (воскресенье) до 6 (суббота));
    * D - сокращенное наименование дня недели, 3 символа (от Mon до Sun);
    * W - порядковый номер недели года по ISO-8601, первый день недели - понедельник (42);
    * j - день месяца без ведущих нулей (от 1 до 31);
    * d - день месяца, 2 цифры с ведущими нулями (от 01 до 31);
    * m - порядковый номер месяца с ведущими нулями (от 01 до 12);
    * n - порядковый номер месяца без ведущих нулей (от 1 до 12);
    * z - порядковый номер дня в году (нумерация с 0) (от 0 до 365);
    * M - сокращенное наименование месяца, 3 символа (от Jan до Dec);
    * F - полное наименование месяца (от January до December);
    * y - номер года, 2 цифры (99, 03);
    * Y - порядковый номер года, 4 цифры (1999, 2003);
    * O - разница с временем по Гринвичу в часах (+0200);
    * L - признак високосного года. 1, если год високосный, иначе 0;
    * I - признак летнего времени. 1, если дата соответствует летнему времени, иначе 0;
    * t - количество дней в месяце (от 28 до 31);
    * c - дата в формате ISO 8601 (2004-02-12T15:19:21+00:00).
* timestamp - целевая временная метка. Если не указано, используется текущее время.

Возвращаемое значение
=====================

Форматированное значение даты/времени.

Пример использования
====================

Вывод даты и времени в формате ISO 8601:

    echo date('Y-m-d\TH:i:sO');
